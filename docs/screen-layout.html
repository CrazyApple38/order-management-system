5.1.1 画面構成

既存のExcel業務管理計画書と同様のレイアウトを採用し、受注一覧・受注詳細・編集・人員配置を1つの画面で操作可能にする。
左側に現場情報（固定列）、中央に配置ゾーン、右側に社員リスト（サイドパネル）を配置。
社員は右サイドパネルから配置ゾーンへドラッグ&ドロップで配置する。
現場情報セルをクリックすると、時間・会社・現場名・区分・昼夜の入力ポップアップが開く。

5.1.2 レイアウト詳細

ヘッダー部

  • 日付選択（日付ピッカー）
  • 前日/翌日ボタン
  • グループ会社フィルタ（ドロップダウン）

ツールバー

  • 受注簿データ取得
  • 受注簿へ転記
  • 行挿入/削除/消去
  • 行の並べ替え（↑↓）
  • フィルタ/空行非表示

左側固定列

  • No.（行番号）
  • 時間/会社/現場名（クリックで編集ポップアップ）
  • 　※区分・昼夜も同じポップアップで編集
  • 地図（クリックでURL入力ポップアップ）
  • 集合時間
  • 人数（必要/配置済み）
  • 配置（車両・送迎は下部に表示）

配置ゾーン

  • ドロップ可能なエリア
  • 配置された社員をタグ表示
  • ×ボタンで配置解除
  • インジケーター表示

右サイドパネル（社員リスト）

  • 全社員の略称を表示
  • 検索機能
  • ドラッグ&ドロップで配置
  • 配置済みは緑色タグ

5.1.6 セル操作

社員タグをドラッグ 右サイドパネルから社員タグを配置ゾーンへドラッグ&ドロップ
×ボタンクリック 配置済み社員の×ボタンで配置を解除
現場セルクリック 時間/会社/現場名/区分/昼夜の入力ポップアップを開く
地図セルクリック Google MapsのURL入力ポップアップを開く（MAPで表示）
行ダブルクリック 配置詳細ダイアログを開く（移動手段、備品等）
右クリック コンテキストメニュー（編集、削除、コピー等）
