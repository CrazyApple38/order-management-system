# Order Management System

受注管理システム - Supabase + Next.js

## 概要

Excel VBAで運用していた受注簿・業務管理計画書をWebシステム化したプロジェクト。

## 技術スタック

- **Frontend**: Next.js 14+ (App Router)
- **Backend**: Supabase
  - Database: PostgreSQL
  - Auth: Supabase Auth
  - Storage: Supabase Storage
- **Styling**: Tailwind CSS (予定)
- **Language**: TypeScript

## ディレクトリ構成

```
order-management-system/
├── docs/                 # ドキュメント
│   └── 01_要件定義.md
├── src/                  # ソースコード（Next.js）

```
